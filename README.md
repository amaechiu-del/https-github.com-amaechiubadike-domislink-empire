## рҹҡҖ QUICK START

### 1. Clone & Install
```bash
cd domislink-empire
npm install
```

### 2. Setup Environment
```bash
cp .env.example .env.local
# Fill in your API keys
```

### 3. Setup Database
- Go to Supabase
- Create project
- Run `database.sql` in SQL Editor

### 4. Run Development
```bash
npm run dev          # All apps
npm run dev:hub      # Just landing page
npm run dev:realestate
npm run dev:tickets
npm run dev:teachmaster
npm run dev:admin
```
